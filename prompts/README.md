# Prompts

Curated collection of prompts for AI agents and systems.

## Structure

Organized by capability or domain:
- `classification/` — Categorization and classification prompts
- `extraction/` — Data extraction and structured output
- `generation/` — Content generation prompts
- `reasoning/` — Chain-of-thought and reasoning prompts

## Guidelines

- Document the expected input/output format
- Include examples for few-shot scenarios
- Note any model-specific considerations
